# Security Policy

Please do not open a public issue for vulnerabilities, exposed credentials, authentication bypasses, or reports that contain private user data.

Send a private message to [@SmetankaKluss on Telegram](https://t.me/SmetankaKluss) with the prefix `MapKluss security`. Include the affected URL or component, reproduction steps, expected impact, and a safe way to contact you. Do not send real user credentials or access tokens.

General bugs and feature requests belong in [GitHub Issues](https://github.com/SmetankaKluss/mapartforge/issues).
