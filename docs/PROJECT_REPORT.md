# MapKluss — Project Report

## 1. О проекте и его цель

**MapKluss** (mapkluss.art) — бесплатный браузерный генератор map art для Minecraft.
Конвертирует любое изображение в схему или файл карты, которые можно построить
в Minecraft через мод Litematica или в ванильном survival-режиме.

**Цель:** упростить жизнь билдерам — дать инструмент без регистрации, без ограничений,
с профессиональным качеством дизеринга прямо в браузере.

Репозиторий: https://github.com/SmetankaKluss/mapartforge  
Сайт: www.mapkluss.art (домен mapkluss.art настраивается)

## 2. Что реализовано

### Обработка изображений
- 7 алгоритмов дизеринга: None, Floyd–Steinberg, Stucki, JJN, Atkinson, Blue Noise, Yliluoma #2, **KlussDither** (собственный)
- Режим сравнения двух алгоритмов (split-view)
- Панель Adjustments: яркость, контраст, насыщенность (с кнопкой ON/OFF — при OFF эффекты отменяются)
- Поддержка PNG с прозрачностью: режимы Fill BG (с выбором цвета) и Transparent (воздух в экспорте)

### Режимы арта
- **2D Flat** (~61 цвет) — плоская схема
- **3D Staircase** (~183 цвета) — лестничная схема с тремя оттенками (dark/mid/bright)

### Экспорт
- **PNG** — превью картинки
- **.litematic** — схема для мода Litematica (одна кнопка, режим берётся из превью)
- **.dat** — файл карты для ванильного Minecraft
- **ZIP** — весь арт побит на тайлы 128×128, каждый как отдельная литематика
- Барьерные блоки из 3D-схем удалены полностью

### Палитра и блоки
- Полная настройка палитры: включение/выключение групп и отдельных блоков
- Пресеты палитры, сохранение и share-ссылка на палитру
- Support-блок для 3D: выбор материала + 3 режима опоры (под floating-only / под каждый / 2 блока под каждым)
- Shade picker в BlockPickerPopup (в 3D-режиме выбор dark/mid/bright при замене блока)

### Редактирование
- Инструменты: пипетка, кисть (размер 1/2/3), заливка, undo/redo (до 20 шагов)
- Repaint: замена цвета блока через клик на пиксель
- Список материалов с подсчётом в стаках и шалкерах

### UI / UX
- Интерактивный тур (12 шагов, driver.js)
- Wiki-модал с 10 разделами документации
- Адаптив для мобильных (вкладки Settings / Palette / Export)
- Масштабирование интерфейса до 1600px (поддержка 2K мониторов)
- Сохранение настроек в localStorage
- Share-ссылка (настройки + изображение через внешнее хранилище)
- Crop-инструмент для обрезки изображения под соотношение сетки

**Ключевые решения:**
- OKLAB вместо RGB для дистанций цветов — заметно лучше визуальный результат
- Web Worker для обработки — UI остаётся отзывчивым на больших изображениях
- KlussDither — гибридный алгоритм (Stucki + snap зоны + serpentine) для чистых краёв
- `bgMode` в ProcessOptions — PNG прозрачность прокидывается до worker'а без изменения всех callsite'ов (через ref)
- Барьерные framing-блоки удалены из литематик — были не нужны
