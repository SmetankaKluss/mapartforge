MapKluss — Changelog
====================

v1.9.0
------
- Мобильная вкладка canvas + онбординг-стрелка для новых пользователей
- Анимации пустого состояния, превью алгоритмов дизеринга, progress bar в стиле Minecraft
- Устранение лага split line: прямое обновление DOM без React re-render
- Устранение лага hover tooltip: позиция обновляется императивно, re-render только при смене блока
- Устранение лага превью инструмента Select: рисование overlay синхронно в mousemove

v1.8.0
------
- Фильтрация палитры по версии Minecraft (1.12.2 → 1.21.4): недоступные блоки полностью скрываются
- Добавлены блоки версии 1.21.4: Pale Oak (planks, slab, log, stripped log, pressure plate),
  Resin Clump, Resin Block, Tuff Bricks, Tuff Brick Slab, Polished Tuff, Polished Tuff Slab, Chiseled Tuff
- Кнопка скачивания материалов перенесена в список MaterialsList с учётом переключателя макс/карта
- Селектор версии MC перемещён выше к панели палитры
- Панель экспорта сворачивается по клику на заголовок

v1.7.0
------
- История проектов на базе IndexedDB: сохранение и загрузка проектов
- Глобальный drag-and-drop изображений в любое место страницы с full-screen оверлеем

v1.6.0
------
- Полная локализация интерфейса RU/EN (i18n)
- Масштабирование скроллом (Ctrl+Scroll / Shift+Scroll)
- Многошаговое уменьшение изображения для лучшего качества
- Устранение лага кисти: позиция курсора через imperative DOM, без re-render на каждый mousemove
- Улучшение визуального дизайна UI

v1.5.0
------
- Инструмент «Паттерн» (P): тайловая кисть с редактором паттерна (перетаскиваемый попап),
  ползунок размера кисти
- Инструмент «Градиент» (G): drag для задания направления, цветовые стопы,
  интерполяция в OKLab, дизеринг Байера 4×4

v1.4.0
------
- Инструменты выделения: прямоугольник (R), лассо (L), волшебная палочка (W), пиксель
- Операции над выделением: удалить, залить, инвертировать, перенести на новый слой
- Горячие клавиши: Ctrl+A / Ctrl+D / Ctrl+I, Del, Esc

v1.3.0
------
- Две кнопки экспорта в artist mode: LITEMATIC (все слои) и СЛОЙ (текущий)

v1.2.0
------
- Настройки по слоям: mapMode, staircaseMode, dithering
- Гибридный экспорт litematic (3D + 2D слои в одном файле)

v1.1.0
------
- Artist mode: слои, кисть, ластик, заливка, текст, изменение прозрачности

v1.0.0
------
- Базовый запуск: дизеринг (KlussDither), палитра 352 блока, экспорт PNG / litematic
