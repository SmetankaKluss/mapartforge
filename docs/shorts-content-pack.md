# MapKluss Shorts/TikTok Content Pack

## UTM template

Use one link per platform and video:

```text
https://mapkluss.art/?utm_source={platform}&utm_medium=short_video&utm_campaign=mapkluss_launch&utm_content={video_slug}
```

Platforms:
- `youtube_shorts`
- `tiktok`
- `vk_clips`
- `instagram_reels`

## 10 video scripts

### 1. MapartCraft vs MapKluss
- Hook: `MapartCraft уже устарел? Я сделал свой map art generator.`
- Record: old-style workflow reference screen, MapKluss upload, dithering switch, 3D preview, Litematic export.
- Overlay: `MapartCraft alternative -> 3D preview -> Litematic`
- CTA: `Try it: mapkluss.art`
- UTM content: `mapartcraft_vs_mapkluss`

### 2. Image -> map art -> Litematic
- Hook: `Картинка превращается в Minecraft map art за 30 секунд.`
- Record: upload image, choose size, process result, open export, download Litematic.
- Overlay: `Image -> Map Art -> .litematic`
- CTA: `Собери свой арт на mapkluss.art`
- UTM content: `image_to_litematic`

### 3. 2D vs 3D Stair
- Hook: `2D map art против 3D stair: разница видна сразу.`
- Record: same image in 2D, switch to 3D Stair, open schematic preview, toggle side view.
- Overlay: `2D Flat / 3D Stair / Height preview`
- CTA: `Проверь свой арт до постройки`
- UTM content: `2d_vs_3d_stair`

### 4. Preview like in Minecraft
- Hook: `Теперь можно посмотреть арт почти как в Minecraft, не заходя в игру.`
- Record: generated art, open preview, switch Scene, click gallery/nether/house, download screenshot.
- Overlay: `Scene preview / Gallery / Nether / House`
- CTA: `mapkluss.art`
- UTM content: `minecraft_scene_preview`

### 5. MAP.DAT export
- Hook: `Нужен map.dat, а не только схематика?`
- Record: upload image, process, export MAP.DAT, show downloaded file.
- Overlay: `Minecraft map.dat export`
- CTA: `Генерируй map.dat в браузере`
- UTM content: `map_dat_export`

### 6. Materials list for survival
- Hook: `Перед стройкой лучше узнать, сколько блоков понадобится.`
- Record: generate 3D art, open materials list, toggle per-map mode, download materials.
- Overlay: `Materials / Per map / Survival planning`
- CTA: `Список материалов на mapkluss.art`
- UTM content: `materials_survival`

### 7. Large multi-map art
- Hook: `Один арт на 3x3 карты выглядит совсем иначе.`
- Record: switch 1x1 to 3x3, process a detailed image, zoom preview, export.
- Overlay: `1x1 -> 3x3 / 384x384 blocks`
- CTA: `Попробуй большой map art`
- UTM content: `large_multimap_art`

### 8. Anime/logo/photo conversion
- Hook: `Один инструмент, три разных типа арта.`
- Record: fast cuts: logo, anime frame, photo; show different dithering choices.
- Overlay: `Logo / Anime / Photo / Dithering matters`
- CTA: `Загрузи свою картинку`
- UTM content: `anime_logo_photo`

### 9. Before/after dithering
- Hook: `Дизеринг решает, будет арт чистым или шумным.`
- Record: same image with None, Floyd-Steinberg, Stucki, Blue Noise, KlussDither.
- Overlay: `Same image / Different dithering`
- CTA: `Сравни режимы сам`
- UTM content: `dithering_before_after`

### 10. Build tracker workflow
- Hook: `Если строите арт командой, нужен трекер.`
- Record: generate art, open build tracker, show material checklist and placed blocks UI.
- Overlay: `Map art -> Materials -> Build tracker`
- CTA: `Собирайте арт без хаоса`
- UTM content: `build_tracker_workflow`
