MapKluss — Changelog
====================

v1.13.0
-------
- Все страницы вне редактора переведены на единый sharp workshop-дизайн MapKluss с общей адаптивной шапкой
- Галерея заменена на четыре честных source→result-примера: собственный логотип и три работы public domain с атрибуцией
- Примеры, SEO-гайды, Cloud, Device Approval, страницы арта/коллекции и Build Tracker получили единую типографику, навигацию, состояния и улучшенную доступность
- Гостевой Cloud получил полноценный экран знакомства со связкой редактор → облако → Minecraft, единый блок входа и скачивания мода, а также корректную мобильную прокрутку
- Подготовленные SEO-гайды остаются скрытыми до отдельного запуска: ссылки, CTA и sitemap-записи на них удалены
- Статические маршруты загружаются без ожидания Supabase; добавлены ясные loading/error-состояния

v1.12.0
-------
- Полностью обновлена система иконок: Radix для микроконтролов, Tabler для инструментов и Pixelarticons для крупных игровых акцентов
- Убраны устаревшие Iconify-пакеты, системные emoji и шрифтовые Unicode-глифы из интерактивных элементов
- Иконки получили согласованные размеры 15/18/24 px, корректные смыслы и доступные подписи для icon-only кнопок
- Экспорт, аккаунт, облако, слои, палитра, модальные окна и трекер используют собственные семантические пиктограммы

v1.11.0
-------
- Боковые панели настроек и палитры можно независимо сворачивать; состояние сохраняется
- Плавный масштаб 50–800% с фокусом под курсором по Ctrl/⌘ + колесу
- Инструмент-курсор перемещает увеличенный арт по рабочей области перетаскиванием ЛКМ; подсказка блока при этом сразу скрывается
- Яркость, контраст, насыщенность и RGB получили кнопки ±1 с ускорением при удержании

v1.10.1
-------
- Визуальный редизайн: цельные Iconify-иконки, улучшенные панели, отступы и контраст
- Повышена мобильная читаемость интерфейса

v1.10.0
-------
- Импорт и экспорт MAP.DAT
- GIF-проекты и GIF LITEMATIC PACK
- Трекер постройки для команды и восстановленные публичные share links

v1.9.0
------
- Мобильная вкладка canvas + онбординг-стрелка для новых пользователей
- Анимации пустого состояния, превью алгоритмов дизеринга, progress bar в стиле Minecraft
- Устранение лага split line: прямое обновление DOM без React re-render
- Устранение лага hover tooltip: позиция обновляется императивно, re-render только при смене блока
- Устранение лага превью инструмента Select: рисование overlay синхронно в mousemove

v1.8.0
------
- Фильтрация палитры по версии Minecraft (1.12.2 → 1.21.4): недоступные блоки полностью скрываются
- Добавлены блоки версии 1.21.4: Pale Oak (planks, slab, log, stripped log, pressure plate),
  Resin Clump, Resin Block, Tuff Bricks, Tuff Brick Slab, Polished Tuff, Polished Tuff Slab, Chiseled Tuff
- Кнопка скачивания материалов перенесена в список MaterialsList с учётом переключателя макс/карта
- Селектор версии MC перемещён выше к панели палитры
- Панель экспорта сворачивается по клику на заголовок

v1.7.0
------
- История проектов на базе IndexedDB: сохранение и загрузка проектов
- Глобальный drag-and-drop изображений в любое место страницы с full-screen оверлеем

v1.6.0
